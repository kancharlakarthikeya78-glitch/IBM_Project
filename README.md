# AI-Based Smart Water Conservation Advisor

Website link: https://web-designer-ai--kancharlakarth1.replit.app/

This project is developed as part of the **1M1B – IBM SkillsBuild AI for Sustainability Virtual Internship**.

## SDG Alignment
- **Primary SDG:** SDG 6 – Clean Water and Sanitation
- **Secondary SDG:** SDG 12 – Responsible Consumption and Production

## Project Overview
Water wastage is a common issue in college campuses due to lack of awareness and guidance.  
This project uses AI-based decision-making to analyze daily water usage habits and provide personalized water conservation recommendations.

## AI Technologies Used
- **Agentic AI:** Rule-based decision logic to analyze user inputs and generate appropriate recommendations.
- **Retrieval-Augmented Generation (RAG):** Conceptual integration to retrieve verified water conservation guidelines before generating advice.
- **IBM Granite:** Conceptual large language model used to generate natural-language responses.
- **Prompt Engineering:** Ensures accurate and responsible AI behavior.

## Features
- User input for daily water usage
- AI-powered water-saving recommendations
- Backend API for advice generation
- Scalable architecture for future enhancements

## Responsible AI
- No personal or sensitive data is collected
- Transparent decision logic
- Educational and sustainability-focused usage
